# codecode

Python coding practice workspace.

## Workflow

1. Open the current problem under `problems/`.
2. Implement the function in that problem's `solution.py`.
3. Run:

```bash
uv run pytest problems/001-running-sum -q
```

Ask Codex to grade when tests pass or when you want review.
